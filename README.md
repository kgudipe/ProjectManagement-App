# ProjectManagement-App

A lightweight project management app to plan work, track tasks, and keep teams aligned. It covers the basics: create projects, add tasks, update status, and view progress in one place.

## Hosted Link
https://main.d3im8sx6kpdu7p.amplifyapp.com

## How to Use
- Open the app and create a project.
- Add tasks with a title, details, and status.
- Move tasks across statuses as work progresses.
- Edit or delete tasks when priorities change.

## Future Scope
- Team invites and role-based access.
- Task comments, attachments, and activity logs.
- Analytics dashboards for velocity and bottlenecks.
- Notifications and integrations (Slack, email, calendar).
